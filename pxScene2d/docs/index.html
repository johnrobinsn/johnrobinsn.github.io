pxScene - Scene Graph API

A 2d scene graph API supporting the following:
  • 2d hierarchical affine scene graph.
  • Small set of atomic UI elements
  • Complex components built through composition.
  • animation primitives
  • rich masking primites
  • more to come...

Object instance

Properties

parent
returns this object's parent Object
x
x-coordinate used as input into the object's transform function in pixel units
y
y-coordinate used as input into the object's transform function in pixel units
w
pixel unit width of the object
h
pixel unit height of the object
cx
x offset used as the center of rotation and scale
cy
y offset used as the center of rotation and scale
sx
scale factor in the x dimension
sy
scale factor in the y dimension
a
opacity of the object [0-1]
r
angle of rotation in degrees
id
associated string identifier for the object
interactive
determines whether the object is mouse interactive. defaults to true.
painting
when set to false the object and it's children will be snapshotted. when set to true the object and it's children will immediately reflect any changes
clip
determines whether the drawing done by the object and it's children will be clipped by the objects w and h properties. defaults to false.
mask
specifies a url for an image to use as this object's alpha mask
drawAsMask
determines whether this object will be used to define an alpha layer mask for the siblings of this object. defaults to false.
draw
determines whether this object will be drawn
numChildren
returns the number of immediate child objects that this object parents
children
returns a collection of child objects for this object

Methods

getChild(i)
returns the i-th child for this object
remove
removes this object from the scene
removeAll
removes all children and their descendents from the scene.
animateTo(json, duration, tween, type)
tween the object's properties to the targets specified over the specified duration using the tween function. type determines whether the animation ends, loops or oscillates (seesaw). returns a promise that will fulfill when the animation is complete.
on(string, function)
registers a function callback for the specified event
delListener(string, function)
unregisters a function callback for the specified event

TextBox instance inherits from Text

Properties

wordWrap
when width > 0, and wordWrap is true, wraps text at width of text box
xStartPos
x position relative to the text box object, in pixels, for the first line of text only, similar to an indent
xStopPos
x position where text must stop flowing on the last line of text only. If ellipses are applied, the ellipses must also stop at this location.
NOTE: xStartPos and xStopPos are only applied when horizontalAlign=LEFT.
NOTE: xStopPos can only be applied when truncation is true.
ellipsis
indicates if ellipsis shall be shown when truncation occurs. Ignored if truncation=NONE.
truncation
indicates what type of truncation is to be applied to the text if it does not fit within the text box dimensions.
  • NONE - no truncation (constant value: 0)
  • TRUNCATE - text is truncated at the bottom of the text box object. The last word may be partially truncated. (constant value: 1)
  • TRUNCATE_AT_WORD_BOUNDARY - text is truncated at the bottom of the text box object. Truncation occurs at the word boundary. (constant value: 2)
    • alignHorizontal
    vertical alignment to be applied to text within the text box dimensions
  • TOP - text is vertically aligned at the top. Default. (constant value: 0)
  • CENTER - - text is vertically aligned at the center. (constant value: 1)
  • BOTTOM - text is vertically aligned at the bottom. (constant value: 2)
    • alignVertical
    horizontal alignment to be applied to text within the TextBox dimensions
  • LEFT - text is horizontally aligned to the left. Default. (constant value: 0)
  • CENTER - text is horizontally aligned to the center. (constant value: 1)
  • RIGHT - text is horizontally aligned to the right. (constant value: 2)
    • leading
    number of pixels of height to apply between lines of text when multiple lines of text exist. May be negative.
    getFontMetrics
    Returns information about the font face (font and size). It does not convey information about the text of the font. See section 3.a in http://www.freetype.org/freetype2/docs/tutorial/step2.html. The returned object has the following properties:
    (Soon to be deprecated.)
  • height - float - the distance between baselines
  • ascent - float - the distance from the baseline to the font ascender (note that this is a hint, not a solid rule)
  • descent - float - the distance from the baseline to the font descender (note that this is a hint, not a solid rule)
  • naturalLeading – float – the distance that must be placed between two lines of text (See linegap in section 2 in http://www.freetype.org/freetype2/docs/glyphs/glyphs-3.html#section-1)
  • baseline – float – the baseline position for the first line of text in this widget
    • measureText
    returns an object with the following properties (measurements are relative to (x,y) of the TextBox object):
  • bounds - object - {x1:0, y1:0, x2:0, y2:0} - The two points representing the bounding rectangle of the complete text
  • charFirst - {x:0, y:0} - The x position represents the left most rendered pixel of the first character on the first line of text. The y position represents the baseline.
  • charLast - {x:0, y:0} - The x position represents the right most rendered pixel of the last character on the last line of text. The y position represents the baseline.
